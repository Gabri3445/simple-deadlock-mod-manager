# Simple Deadlock Mod Manager

A lightweight mod manager for **Deadlock** that makes installing and managing mods easy.

## Features

- [x] Load `.vpk` mod files
- [x] Assign custom names to mods
- [x] Automatic **Deadlock** path detection
- [x] Automatically fixes/updates `gameinfo.gi` to keep it valid
- [x] Enable and disable mods easily

## Planned Features

- [ ] Load `.vpk` files directly from `.zip` archives
- [ ] Download mods from **GameBanana**

## Not Planned

- Automatic mod updates

## Platform Support

- **Linux:** Tested
- **Windows:** Not tested, but should work.
